## Introduction
Promptopia is an open-source AI prompting tool for the modern world to discover, create and share creative prompts.

In the application, you can find a Prompt made by the other user that you can use in the Chat GPT or any AI application. And if you want to share an AI Prompt you can sign in with your Google Account and then Create a Prompt üëç

### Tech Stack üíª :
- React
- NextJS
- TailwindCSS
- MongoDB

### Cloning the repository

```shell
git clone https://github.com/inifarhan/promptopia.git
```

### Install packages

```shell
npm i
```

### Setup .env file


```js
MONGODB_URI=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
NEXTAUTH_URL=
NEXTAUTH_URL_INTERNAL=
NEXTAUTH_SECRET=
```

### Start the app

```shell
npm run dev
```
